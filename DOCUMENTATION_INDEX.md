## ⚡ 30-Second Summary

**SafeRoute AI** is India's first AI-powered women's safety commute assistant that:

1. **Predicts** route safety before you travel using Random Forest AI
2. **Detects** voice distress automatically (English, Hindi, Tamil)
3. **Responds** to emergencies in under 3 seconds

**Tech**: React + Node.js + MongoDB + Flask AI services
**Status**: ✅ Production-ready, hackathon-winning application
